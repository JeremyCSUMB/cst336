Windows is by far the most dominant operating system in the realm of personal computers.

It is estimated that around 90% of all computers run on Windows, so this is quite a bit more than Linux and MacOS users.

Windows is the first personal computer operating system that featured a graphical user interface.

Quick facts about Windows

  • The 1995 release of Windows featured Windows own internet brower, Internet Explorer.
  • Originally made for IBM machines, windows now run on hundreds of millions of devices worldwide.
  • Window's AI assistant, Cortana, rivals with Apple's Siri and Amazon's Alexa.
